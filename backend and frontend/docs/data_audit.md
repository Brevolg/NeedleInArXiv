## Ограничение валидности

Gold answer и answer facts пригодны для последующей RAG-оценки ответа, но текущий проект
реализует retrieval. Он оценивает попадание документов, а не генерацию ответа. Для полноценной
оценки RAG нужен отдельный генератор и проверка полноты фактов.
